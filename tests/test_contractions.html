Test French Contractions

Hover over the purple words to test contraction removal and lookup.

How contractions work:
French contracts certain words before vowels. When these appear, the extension should:
  1. First try to find the word WITH the contraction (some expressions start with l', d', etc.)
  2. If not found, remove the contraction and try again
  3. Display the definition of the de-contracted word

4. Other contractions
